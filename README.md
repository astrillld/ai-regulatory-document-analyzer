# AI Regulatory Document Analyzer

A Python project for parsing regulatory PDF documents, chunking text, building semantic search with embeddings, and detecting requirement-related passages.

## Planned features
- PDF text extraction
- Text chunking
- Embedding-based semantic search
- Requirement detection
- Query-based retrieval

## Tech stack
- Python
- PyMuPDF
- sentence-transformers
- FAISS
- pandas / numpy
- pytest
